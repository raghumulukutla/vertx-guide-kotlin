Kotlin+Gradle implementation of https://github.com/vert-x3/vertx-guide-for-java-devs

== Building

To launch your tests:
```
./gradlew clean test
```

To package your application:
```
./gradlew clean assemble
```

To run your application:
```
./gradlew clean run
```
